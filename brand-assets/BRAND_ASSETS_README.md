# Line Coffee — Brand Assets

This folder contains all visual brand assets for Line Coffee V2.

## Rules

- Do not delete original logo files.
- Always keep original high-resolution files.
- Website-ready images should be optimized before production.
- Product images should have both edited and transparent PNG versions when needed.
- Brand colors:
  - Dark Brown: #522500
  - Beige: #FFDCC2
  - Black: #000000
  - White: #FFFFFF
- Fonts:
  - English: Playfair Display
  - Arabic: Cairo

## Folder Usage

- logo: official logo files and favicon versions.
- packaging: packaging and sticker visuals.
- product-photos: real product photography.
- website-visuals: hero images, banners, textures, section visuals.
- social-media: posts, reels, stories, ads.
- references: inspiration and visual references only.

## Important

Files inside references are inspiration only.
Do not use reference images directly as final commercial assets unless properly licensed.
